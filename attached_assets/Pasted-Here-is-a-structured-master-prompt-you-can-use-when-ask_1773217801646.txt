### One thing most founders miss (important)

Most web apps fail on mobile because **they try to shrink desktop UI instead of redesigning flows**.

Your travel app should likely change:

| Desktop Pattern     | Mobile Pattern        |
| ------------------- | --------------------- |
| Data tables         | Card lists            |
| Multi column layout | Single scroll layout  |
| Sidebar nav         | Bottom tab nav        |
| Toolbar actions     | Sticky bottom actions |
| Popups              | Bottom sheets         |
