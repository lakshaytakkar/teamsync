# Mobile UX Rules for These Components

Important rules to enforce:

### Tap Size

Minimum **44–48px touch area**

### Spacing

Minimum **8–12px between actions**

### List Rows

Minimum height **56px**

### Bottom Actions

Primary actions should sit **near thumb zone**

### Gestures

Prefer:

* swipe actions
* bottom sheets
* pull refresh

instead of complex menus.
